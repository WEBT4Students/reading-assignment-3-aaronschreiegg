Lists and Tables

HTML for lists

unordered list: starts with the <ul> tag. Each list item starts with the <li> tag.
ordered list: starts with the <ol> tag. Each list item starts with the <li> tag.

A description list is a list of terms, with a description of each term.
The <dl> tag defines the description list
The <dt> tag defines the term (name)
The <dd> tag describes each term.

A nested list is a list inside a list.

CSS for lists

Make a bullet a square:
ul.b {
     list-style-type: square;
      }

Make bullet an image named "my-bullet.png":
ul.b {
      list-style-image: url ("my-bullet.png");
     }

Make bullet an uppercase roman letter":
<ol style ="I">

Set bullet position to be inside the list item's text flow:
ul.b {
      list-style-position: inside;
      }

Set bullet position to be outside the list item's text flow
ul.b {
      list-style-position: outside;
      }
